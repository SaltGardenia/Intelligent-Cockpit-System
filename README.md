# Intelligent-Cockpit-System（智能座舱系统）
## 👋欢迎访问此项目

---

### 😀主要功能：

- 行车记录
- 驾驶员状态检测
- 音乐播放
- 导航
- 电话拨号

---

### 😁开发环境：

- Python 3.10

- 包管理：pip, conda（详见requirements.txt）

- 数据库：MySQL

- GUI：PyQt

- 封装：docker

---

### 🤔功能详述：

#### 登录、注册：

- 使用MySQL提供数据交互支持。
- 对个人密码进行MD5加密。

#### 驾驶检测：

- 通过yolo框架搭建人脸识别模型，对驾驶员行为进行实时检测，并录像保存，可人为主动进行拍照。
- 对驾驶员的姿态分类有正常驾驶、闭眼、打哈欠、打电话、左顾右盼等5种行为。

- 录像和照片会保存至相关的列表。

#### 道路检测：

- 实时播放行车画面。

#### 音乐：

- 音乐播放界面集成了音乐播放的相关功能（播放、暂停、切歌、播放模式、歌单列表管理）。

#### 地图：

- 调用百度地图API，结合PyQt的QtWebEngineWidgets加载网页地图。

#### 拨号：

- 拨号界面。
